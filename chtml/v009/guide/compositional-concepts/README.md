# Compositional Concepts

This section covers how to define components for later use. We start by answering where to place these components – especially important as we might need to work with a handful of them. Then we’ll see different ways to reuse components.

The essence of componentization is reusability and composability. CHTML makes it easy to define once and use everywhere.
